Ubuntu 18.04 下安装shadowsocks

发表于 2019-01-17 | 分类于 Linux

方法

1. 下载所需工具

pip 和 shadowsocks

sudo apt-get install python-pip
sudo pip install shadowsocks

缺什么下什么就对了

2. 配置

sudo emacs /etc/shadowsocks.json

{
    "server":"my_server_ip",
    "server_port":8388,
    "local_address": "127.0.0.1",
    "local_port":1080,
    "password":"mypassword",
    "timeout":520,
    "method":"aes-256-cfb",
}
